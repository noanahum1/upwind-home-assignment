# 🛡️ Upwind Home Assignment 

## 📌 Overview
This repository contains three security-focused projects demonstrating phishing detection, malware analysis, and SQL injection vulnerabilities.

Each part is independent and includes its own README with full explanation and setup instructions.

---

## 📂 Repository Structure

### Part 1 – Phishing Detector  
Client-side web application that analyzes email content to detect phishing attempts using pattern recognition and fuzzy matching.  
👉 https://github.com/noanahum1/upwind-home-assignment/tree/main/part1-phishing-detector

---

### Part 2 – Malware Analysis Sandbox  
Sandbox environment that executes simulated malware inside Docker and analyzes its behavior through logs.  
👉 https://github.com/noanahum1/upwind-home-assignment/tree/main/part2-Malware-Analysis-Sandbox

---

### Part 3 – SQL Injection Demo  
Demonstrates a vulnerable login flow and a secure implementation using parameterized queries.  
👉 https://github.com/noanahum1/upwind-home-assignment/tree/main/part3-SQL-Injection-Attack

---

Each project includes its own setup instructions in its directory.
